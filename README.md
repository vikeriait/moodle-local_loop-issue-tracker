# Kompliance — Public Issue Tracker

This repository is the **public issue tracker** for
**[Kompliance](https://moodledev.io)**, a Moodle plugin that automates the
lifecycle of recurring professional certifications.

> **The source code is in a private repository.** This repo exists solely to
> provide a public, accessible channel for bug reports, feature requests, and
> support questions — as required by the Moodle Marketplace submission
> guidelines.

## What goes here

- 🐛 **Bug reports** — something isn't working as expected
- 💡 **Feature requests** — ideas for new capabilities
- ❓ **Questions / support** — need help configuring or using the plugin

## Before opening an issue

1. **Search existing issues** (open and closed) to avoid duplicates.
2. Use the issue template that matches your case (Bug Report or Feature
   Request).
3. Include your **Moodle version** and **Kompliance version** in every report.

## Documentation

Full user guide and admin documentation are bundled with the plugin
(`docs/` directory) and available on the plugin's documentation site.
